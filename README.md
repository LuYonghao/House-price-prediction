背景描述：<br/>
   - 让购房者描述他们梦想中的房子，他们可能不会从地下室天花板的高度或与东西铁路的接近程度开始。但是这个游乐场比赛的数据集证明，与卧室数量或白色栅栏相比，对价格谈判的影响要大得多。<br/>
   - 凭借 79 个解释变量（几乎）描述了爱荷华州艾姆斯住宅的各个方面，这项竞赛挑战您预测每栋房屋的最终价格。<br/>
  <br/>
  
实际目标：<br/>
   - 清理杂乱的，有缺失的数据，建立有效模型，根据已有房型数据，预测房价。<br/>
  <br/>
<br/>
过程：<br/>

   - 数据转换
   - 特征工程
   - 建模学习
